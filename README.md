# Lab 4 Cloud Computing

## Install modules:
```
npm install request
npm install fs
npm install dotenv --save
```

## To set up:
- Rename `.env.example` to `.env` and put in the `authorization header` and `request links` 
- Create the following files and type in the request bodies:
    - partA.json
    - partB.json
    - partC.json 


## To run script:
```
npm run start
```
